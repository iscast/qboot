# qboot
#### 介绍
qboot开发平台，是基于sprintboot的建立的单体项目，适用于快速开发的单体项目
该平台的功能集成了基本的系统模块(用户/菜单/权限/登陆/任务)，和简单的代码生成。
由开发者扩充业务功能
